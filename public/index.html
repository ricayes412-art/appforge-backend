1

Estructura Web

Tu proyecto debe incluir un archivo index.html en el directorio raíz del archivo ZIP para funcionar.

2

Sube el Archivo

Usa la zona de arrastrar en la pestaña "Nueva Compilación" para cargar el archivo ZIP o el archivo único HTML.

3

Configura la App

Indica el nombre de la app, su versión, ID de paquete (Package ID) y los permisos nativos requeridos.

4

Compilación IA

Presiona el botón "Compilar" y deja que nuestro Agente inteligente complete dependencias de firma.

5

¡Descarga tu App!

Una vez finalizado el pipeline de empaquetado, descarga el APK para Android o el IPA para iOS directamente.

Preguntas Frecuentes

¿Qué tipo de proyectos web puedo compilar?

Puedes compilar cualquier sitio o aplicación web estática estricta construida con HTML5, CSS3, ES2024, JavaScript plano, frameworks de frontend modernos (como React, Vue, Angular, Svelte siempre que compiles a build estático de carpeta dist/out) y Progressive Web Apps (PWAs).

¿Cómo configuro un icono personalizado para mi app?

En la sección "Configuración Avanzada" de la pestaña de Nueva Compilación, puedes subir una imagen en formato .PNG cuadrada (recomendado 512x512 píxeles). Nuestro sistema lo redimensionará y lo adaptará automáticamente para todas las densidades de pantalla en Android e iOS.

¿La APK compilada es apta para subirse a Google Play Store?

Sí. El compilador en la nube de AppForge Cloud firma digitalmente los binarios utilizando las llaves Keystore con algoritmo SHA-256 alineados con las regulaciones de seguridad modernas de Google Play Console y Apple Developer Portal.

Características Técnicas del Compilador

Tecnologías Híbridas de Frontend Soportadas HTML5 estándar, CSS3 avanzado, JavaScript ES2024 moderno, Progressive Web Apps (PWA), Capacitor Core, Apache Cordova, frameworks SPA (React, Vue, Angular, Svelte compilados a estático).
Formatos de Entrada Admitidos Carpeta comprimida .ZIP conteniendo index.html, o un archivo único .HTML estático autocontenido.
Formatos de Salida Generados Android Application Package (.APK) compatible con Android 6.0 (API Level 23) en adelante, iOS Application Archive (.IPA) compatible con iOS 14.0 en adelante.
Motor de Análisis y Optimización IA Agentes inteligentes integrados basados en los LLMs avanzados Claude 3.5 Sonnet y GPT-4o, encargados de la reestructuración automática del DOM, corrección de dependencias rotas y resolución de vulnerabilidades.
Infraestructura del Servidor Nodos de compilación escalables de baja latencia construidos sobre Node.js 20, Docker para entornos aislados (Sandbox) y clústeres optimizados de Kubernetes en AWS.
Tiempo de Compilación Promedio Entre 2 a 8 minutos, dependiendo de la complejidad de los archivos de origen y de la cola del servidor seleccionada.
Límite de Carga de Archivos 500 Megabytes por archivo ZIP de origen.
Integración API REST Endpoints disponibles para compilar programáticamente a través de tokens Bearer asignados en los planes Semestrales y de Marca Blanca.
